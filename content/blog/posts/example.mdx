### Line Marker

- code

```markdown
\```js {2,4-5} del={6} add={8} "word" /\bb\w+[ey]\b/ ins="inserted" del="deleted" add={"1. add style text label": 11-14}
function example() {
  console.log('This line is highlighted');
  console.log('This line is not highlighted');
  console.log('This word is highlighted and Reg Exp is also supported, for example: bye boy');
  console.log('You can set range of lines to be highlighted');
  console.log('This line is not highlighted');
  console.log('This line is deleted');
  console.log('This line is not highlighted');
  console.log('This line is new added');
  console.log('These are "inserted" and "deleted" marker types');


  console.log('You can also test long lines to see how they are handled in the code block.' +
    ' When using text label, remember to spare a line for the label itself.');
  console.log('This is the end of the text label');
}
\```
```
